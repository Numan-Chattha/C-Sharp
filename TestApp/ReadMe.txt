*Project Contains Practice Implementation for C#.
*Railway Crossing scenario has been implemented using Events and Delegates in RailwayCrossing(EventsAndDelegates).cs File.
	*An Event is Fired on a Railway Crossing when A train is About to pass through and All Cars Stop on this Event.
	*An Event is Fired again on Railway Crossing when A train has passed through and All Cars log entry in the singleton class Start Moving again on this Event.

*A logger has been Implemented using Singleton pattern to print details of cars stopped at a Railway Crossing in file RailwayCrossingLogger(Singleton).cs
